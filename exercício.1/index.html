Listas ordenadas e não ordenadas

Aula nove

Listas

Podemos criar listas ordenadas usando a tag OL, mas dentro de OL deve se usado LI para dar sentido e para mostrar para a maquina que será escrita uma lista, assim

  1. Acordar
  2. Banhar
  3. fazer algo para comer
  4. Ir trabalhar

Em OL temos aulguns tipos de estilo, só usar a tag TYPE dentro de OL, assim poderá escolher o estilo da lista.

Existem também listas não ordenadas, essas são usadas apenas para organizar mesmo, o famoso Tanto Faz

  • Comprar mesa
  • Comprar cadeira
  • Comprar monitor
  • Comprar mouse pad

Em UL também temos alguns estilos que podemos usar para diferenciar, também usando TYPE, mas em UL temos apenas três tipos de estilo, entre eles temos DISC, CIRCLE e SQUARE.

Misturando listas

Podemos colocar uma lista ordenada dentro de uma lista não ordenadas, basta colocar uma OL e depois uma UL, assim

  1. Limpar a mesa
    • Tirar todos os componentes do setup
    • Pegar e molhar o pano
    • Usar Veja para limpar
  2. Limpar o vento
    • Desmontar o vento
    • limpar peça por peça
    • Montar de novo
  3. Limpar o quarto
    • Levanrtar a cama
    • Subir todos os fios e sapatos
    • Passar pano type

Imagens flexíveis

Você pode abrir esse site em vários dispositivos, a imagem irá se ajustar com o tamanho da sua tela.

Pra fazer tudo isso funcionar use as tags PICTURE, IMG e SOURCE:MIDIA.TYPE 

picture>
    source media="(max-width: 730px)" srcset="imagem/foto-p.png" type="image/png">
    source media="(max-width: 1030px)" srcset="imagem/foto-m.png" type="image/png">
    img src="imagem/foto-g.png" alt="imagem flexível">
/picture>
